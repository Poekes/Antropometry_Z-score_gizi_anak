# AI Agent Instructions

When working on this repository, you **MUST** adhere to the following rules for every change you make:

1.  **Update `install.sh`**: If your changes introduce new dependencies, structural changes, or require new build steps, you must update the `install.sh` script to reflect these requirements.
2.  **Update `README.md`**: If you add new features, modify existing behavior, or change how the application is built/run, you must update the `README.md` documentation accordingly.
3.  **Compile/Build**: Always ensure that the project compiles successfully after your changes. Run the necessary build commands to verify this.
4.  **Commit Per File**: Do not lump all changes into a single large commit. You must make separate commits for each file changed. The commit message should clearly describe the changes made to that specific file.
